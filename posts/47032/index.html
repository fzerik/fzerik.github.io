Hexo创建新博客

创建新博客

在博客目录下使用打开Git Bash 终端，输入以下命令：

hexo new "Post"

或者

hexo n "Post"

会默认使用post布局，然后自动在source_posts目录生成一个.md文件，使用markdown语法进行编写。
打开.md文件显示如下：

title: 主题
date: 2022-12-12 00:00:00

建议作以下更改（具体信息由实际填写）：

title: 主题
date: 2022-12-12 00:00:00
author: -
coverImg: /source/medias/xxx.jpg #图片在/source中的路径位置
top: true
cover: true
password: -
toc: false
mathjax: false
summary: 自定义的文章摘要内容
categories: Markdown
tags:
  - Blog
  - Markdown

注意事项：对新博客命名时如需多个单词进行命名，可使用 hexo new “My New Post” 或者 hexo new My-New-Post。若不加入引号或间隔号，会默认选择最后一个单词命名。建议无论长短均使用 hexo new “New Name”。

More info: Writing

以下为编写完成后将博客部署到服务器的命令：

clean cache

$ hexo clean

Generate static files

$ hexo generate #可简写为 hexo g

More info: Generating

Run server

$ hexo server #可简写为 hexo s

More info: Server

Deploy to remote sites

$ hexo deploy #可简写为 hexo d

More info: Deployment
